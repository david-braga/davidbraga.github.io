Best practices associated with ID and Class

When creating a website we bascially use HTML, which stands for HyperText Markup Langauge, CSS, short for Cascade Style Sheet, and Javascript or JS.

HTML
HTML is the structure of a Web page, it labels the pieces of content the browser displays. In other words, it's the text.

CSS
CSS is the style or the look of the page and with it you can control how the html elements will look in the broswer.

JS
Javascript is what allows the interaction between the user and the web page in a dynamic way.

ID and CLASS: what are they and which one to use?

ID and Class are selectors that can be added to html elements so they can be acessed using CSS and/or JS.

The value to identify an ID must be used once only in the whole project.

You can have as many IDs as you want but its value must be unique to one specific element only.
